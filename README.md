# Smart Night Attendance System (Noctra)

## 🚀 Overview

A smart hostel monitoring system that ensures secure night attendance using geo-fencing, role-based dashboards, and violation tracking.

## 👤 User Roles

* Student
* Caretaker
* Warden
* Chief Warden

## 🔥 Key Features

* Geo-fenced attendance marking
* Permission management system
* Violation detection (Late, Absent, Left Block)
* Caretaker duty assignment system
* Chief Warden analytics dashboard
* Notification system (Firebase ready)

## 🧱 Tech Stack

* Flutter (Frontend)
* Django REST Framework (Backend)
* PostgreSQL (Database)
* Firebase (Notifications)

## 📊 Highlights

* Role-based access control
* Real-time analytics dashboard
* Scalable backend architecture
* Clean modular design

## ⚙️ Setup Instructions

### Backend


cd backend_Django
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver


### Frontend


cd frontend_flutter
flutter pub get
flutter run
